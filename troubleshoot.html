🔧 Mumble Video Call Troubleshooting

⚠️ Important: Make sure you're testing with http://localhost:8000 not file:// URLs for proper permissions.

🚨 Most Common Issues & Solutions

1. Camera/Microphone Permission Issues

Symptoms: Browser doesn't ask for permissions, or shows permission denied errors
Solutions:
  • Open browser settings → Privacy & Security → Camera/Microphone → Allow for localhost
  • Clear browser data and try again
  • Try in incognito/private mode
  • Make sure no other app is using camera/microphone

2. Screen Sharing Not Available on Mobile

Symptoms: Screen share button is disabled or shows error on mobile devices
Explanation:

Screen sharing is NOT supported on mobile browsers (iPhone, Android) due to security and technical limitations.

  • iOS Safari/Chrome: Apple doesn't allow screen sharing in mobile browsers
  • Android Chrome/Firefox: Limited support, most devices don't support it
  • Mobile Apps: You would need a native mobile app for screen sharing
Alternative Solutions for Mobile Users:
  • 🎥 Use camera instead: Point your camera at your screen/document
  • 💻 Switch to desktop: Use a laptop/desktop computer for screen sharing
  • 📧 Share files: Send documents via email or file sharing
  • 📱 Use screen recording: Record your screen and share the video file
  • 🔄 Mirror to computer: Use apps like TeamViewer or Chrome Remote Desktop

3. Agora App ID Issues

Symptoms: Console shows "Invalid App ID" or connection errors
Check: The App ID 3c19963cb8ca48228834ee38bee6e70f might be expired or invalid
  • Visit Agora Console
  • Create a new project and get a fresh App ID
  • Replace the APP_ID in room_rtc.js

4. Network/Firewall Blocking

Symptoms: Can't connect to Agora servers, timeouts
Try:
  • Disable antivirus/firewall temporarily
  • Try on different network (mobile hotspot)
  • Check if corporate firewall blocks WebRTC

5. Browser Compatibility

Symptoms: Features don't work, console errors about missing APIs
Use: Chrome, Firefox, or Edge (latest versions)
Avoid: Internet Explorer, very old browser versions

📋 Step-by-Step Testing Process

Step 1: Basic Setup
  1. Start local server: python -m http.server 8000
  2. Open http://localhost:8000/lobby.html
  3. Open Developer Tools (F12) and watch Console tab
Step 2: First User Test
  1. Enter name: "User1"
  2. Enter room: "5502"
  3. Click "Go to Room"
  4. Look for console messages:
    • ✅ "Current user UID: [timestamp]"
    • ✅ "RTM login successful"
    • ✅ "Successfully joined RTC channel"
  5. Click "Join Stream" and allow camera/microphone
  6. Look for: ✅ "Successfully published audio and video tracks"
Step 3: Second User Test
  1. Open second browser/incognito window
  2. Go to http://localhost:8000/lobby.html
  3. Enter name: "User2"
  4. Enter room: "5502" (same room!)
  5. Watch console for:
    • ✅ "A new member has joined the room"
    • ✅ "Total members in room: 2"
  6. Click "Join Stream"
  7. Both users should see:
    • ✅ "User published: [other_user_uid] Media type: video"
    • ✅ "Creating new video container for user: [other_user_uid]"

🔍 What to Check in Console

Success Messages (should see these):
  • Current user UID: [number]
  • RTM login successful
  • Successfully joined RTC channel
  • A new member has joined the room: [uid]
  • Total members in room: 2
  • Successfully published audio and video tracks
  • User published: [uid] Media type: video
  • Video track started playing for user: [uid]
Error Messages (investigate if you see these):
  • Agora SDK not loaded properly → Check internet connection
  • NotAllowedError → Camera/microphone permission denied
  • NotFoundError → No camera/microphone detected
  • Invalid App ID → App ID expired or wrong
  • No video track found → Stream not properly published

🆘 Still Not Working?

Last Resort Options:
  1. Try Different Agora App ID: Create new project at console.agora.io
  2. Test on Different Device/Network: Mobile phone, different WiFi
  3. Use Alternative: Try a different WebRTC solution like Jitsi Meet
  4. Contact Support: Share console logs with Agora support
